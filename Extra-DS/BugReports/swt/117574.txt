RIGHT_TO_LEFT | DOUBLE_BUFFERED don't get along
build I20051122-0800 When SWT.RIGHT_TO_LEFT | SWT.DOUBLE_BUFFERED are used the gc clipping is Rect(0,0,0,0) during paint. Example: public static void main(String[] args) { Display display = new Display(); Shell shell = new Shell(display, SWT.SHELL_TRIM | SWT.RIGHT_TO_LEFT | SWT.DOUBLE_BUFFERED); shell.addListener(SWT.Paint, new Listener() { public void handleEvent(Event event) { System.out.println(event.gc.getClipping()); event.gc.drawString(&quot;This is broken &quot; + event.gc.getClipping(), 10, 10); } }); shell.open(); while (!shell.isDisposed()) { if (!display.readAndDispatch()) display.sleep(); } display.dispose (); } Doesn't draw anything.
